# BME 110 — Bioinformatics Lab

Course materials and lab work from BME 110 (Bioinformatics) at UC Santa Cruz.

## Contents
- Genome assembly and sequence alignment exercises
- FASTQ quality control analysis
- UCSC Genome Browser ChIP-seq peak calling (NANOG on POU5F1)
- Fiji/ImageJ image analysis scripts (TrackMate, Weka Segmentation)
- SLURM job submission scripts for HPC cluster
- AlphaFold protein structure visualization
- Sanger sequencing AB1 files and analysis
